通用页面组件演示

展示 List Page 和 Input Page 组件的功能和使用方法

List Page Component 功能特性

  • ✅ 数据表格展示
  • ✅ 分页功能
  • ✅ 搜索/过滤
  • ✅ 排序功能
  • ✅ 批量操作
  • ✅ 导出功能
  • ✅ 响应式设计
  • ✅ 加载状态
  • ✅ 空数据状态

Input Page Component 功能特性

  • ✅ 动态表单字段
  • ✅ 多种布局模式 (单列/双列/步骤)
  • ✅ 表单验证
  • ✅ 自动保存
  • ✅ 文件上传
  • ✅ 响应式设计
  • ✅ 加载状态
  • ✅ 自定义组件支持
help 使用说明

List Page 组件使用说明：

  1. 点击表格列标题可以进行排序
  2. 使用搜索框可以过滤数据
  3. 选择复选框可以进行批量操作
  4. 点击操作列的按钮可以编辑或删除记录
  5. 使用分页器可以浏览更多数据

Input Page 组件使用说明：

  1. 表单支持实时验证，必填字段会显示红色星号
  2. 邮箱和手机号有格式验证
  3. 薪资字段只接受数字输入
  4. 可以上传头像文件
  5. 表单支持自动保存（30秒间隔）
  6. 点击重置按钮可以清空表单

开发提示：

  • 打开浏览器控制台查看事件日志
  • 组件配置可以通过 TypeScript 接口进行类型检查
  • 所有事件都会触发相应的回调函数
  • 组件支持响应式设计，可以调整浏览器窗口大小测试
